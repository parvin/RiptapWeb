# Riptap Website

Landing page for [Riptap](https://riptap.com) — a minimalist rhythm-reaction game for iOS.

## Structure

```
├── index.html        # Landing page (hero, features, screenshot carousel)
├── privacy.html      # Privacy Policy
├── terms.html        # Terms of Use
├── css/style.css     # Styles
├── js/main.js        # Carousel, scroll animations, mobile menu
├── images/           # App icon and future assets
├── CNAME             # GitHub Pages custom domain
└── DEPLOYMENT.md     # GitHub Pages + DNS setup guide
```

## Local Preview

Open `index.html` in a browser:

```bash
open index.html
```

## Deployment

Hosted on GitHub Pages at [riptap.com](https://riptap.com). See [DEPLOYMENT.md](DEPLOYMENT.md) for setup instructions.
